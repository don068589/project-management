# Improvements List

> System data file - normal read/write, no approval required.
> Collects system improvement suggestions. Approved changes are executed and logged to README.md changelog.

## Format

| ID | Issue/Idea | Source | Proposed Change | Status |
|------|-----------|------|----------|------|
| | _(none)_ | | | |

## Status Values

- `pending` - Just proposed, awaiting review
- `approved` - Approved, pending execution
- `executed` - Completed, logged in README.md changelog
- `rejected` - Rejected, reason recorded
- `needs-info` - Needs more information
